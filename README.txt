1) Open the following project home folder in the terminal.
2) Execute the following code to generate the RNAtoProteinFasta.java file 

jflex RNAtoProteinConverter.jflex

3) Compile the RNAtoProteinFasta.java file 

javac RNAtoProteinFasta.java

4) Run the RNAtoProteinFasta.class file with the input7.fasta file

java RNAtoProteinFasta input7.fasta

5) Converted Protein sequences & the relevant information are displayed in the terminal
